# AI_Price_Optima – Dynamic Pricing Dataset

## Overview
This dataset is created for a Dynamic Pricing Optimization project using AI/ML.

It helps analyze how product price and inventory affect sales demand.

## Columns
- **product_id** – Unique product ID  
- **price** – Product selling price  
- **sales_quantity** – Units sold (generated)  
- **inventory** – Stock available  
- **date** – Transaction timestamp  

## Use Case
Used for:
- Demand Forecasting  
- Price Optimization  
- Inventory Management  
- Machine Learning Models
